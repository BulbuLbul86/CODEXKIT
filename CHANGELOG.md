# Changelog

## v0.1-alpha

Статус: экспериментальный публичный релиз.

### Основные возможности

- Сбор переносимого Windows-комплекта для dev/Codex-среды.
- Инкрементальное обновление уже собранного комплекта.
- Поиск и снимки локальных Git-репозиториев.
- Перенос локальных настроек инструментов разработки.
- Офлайн-установщики для выбранных приложений, где это поддерживает источник пакета.
- Разделение большого transfer-архива на части для FAT32-носителей.
- Восстановление на новом компьютере через `2-RESTORE-HERE.bat`.

### Безопасность и ограничения

- Safe mode включён по умолчанию и не копирует явные секреты без согласия пользователя.
- Full mode требует явного подтверждения.
- Переносимый комплект может содержать приватные данные, если пользователь выбрал Full mode или добавил личные пути в `custom-paths.json`.
- Codex Desktop из Microsoft Store может не иметь доступного офлайн-установщика.
- Проект рассчитан на Windows и проверяется на Windows PowerShell 5.1+.
- Релиз экспериментальный: перед использованием на важной машине сначала запускай план восстановления.
